# FestivalApp

FestivalApp est le front d'un site web développé Angular qui utilise le framework Angular Material.

## Installation

Vous pouvez récupérer le code sur le git à l'adresse : (https://github.com/tiffany-dumaire/FestivalApp)
Une partie des anciens fichiers du back s'y trouve également dans le sous-dossier backend qui n'est pas utile pour le front.
Vous retrouverez le back de ce site sur le git à l'adresse : (https://github.com/tiffany-dumaire/Backend-Festival-App)

Après avoir récupérer le code du Front de ce site. 
Vous pourrez le lancer dans le terminal en vous rendant dans un premier temps dans le sous-dossier "/frontend" à l'aide de la commande cd. 
Ensuite vous pourrez lancer la commande ng serve et vous retrouverez le front de l'application sur l'url : (http://localhost:4200)
